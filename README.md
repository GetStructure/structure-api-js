# Structure API
User API for accessing Structure data

## Installation
To install the module via NPM:
```bash
$ npm install --save structure-api-js
```

## Usage
To create a new API client:
```javascript
var api = require('structure-api-js');

var client = api.createClient({
  accessToken: '...'
});
```
